# Dharani Portal Replica

**Project Type:** Full-Stack Web Application   
**Tech Stack:** React.js, Node.js, Express.js, PostgreSQL/MySQL, Tailwind CSS, JWT Authentication  

---

## Project Overview

This is a replica of the **Telangana Dharani Portal**, a government digital land records platform.  
The project demonstrates full-stack development, including **frontend UI**, **backend APIs**, and **database integration**.  
It focuses on **user authentication**, **land record management**, and **dashboard features**.  

---

## Features

- **User Registration & Login:** Secure authentication using JWT  
- **Dashboard:** Responsive interface displaying land records  
- **Land Record Management:** Add, view, and manage land records (CRUD operations)  
- **Database Integration:** PostgreSQL/MySQL backend for storing user and land data  
- **Frontend Design:** Built with React.js and styled using Tailwind CSS  
- **API Integration:** RESTful APIs with Express.js backend
